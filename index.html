Research

I have conducted research in information theory and coding theory, including their application in networking systems. I am also exploring theories related to security, quantum information, and AI.

  • Network coding provides a practical and information-theoretic framework for enhancing network communication efficiency. My research includes two primary aspects:
    • Designing low-complexity network coding schemes, such as the BATS code.
    • Redesigning physical layer coding, media access control, transport protocol, congestion control, and other relevant components within the network coding framework.
  • Shift-XOR codes can be encoded using (non-cyclic) shift and XOR operations, and decoded using a successive cancellation algorithm that incurs nearly the same computation cost as encoding. This technique shows potential for reducing computation costs compared to finite-field operations in various coding problems.
    • We have designed shift-XOR storage codes, regenerating codes, network codes, and more.
    • We are developing a theory of utilizing shift and XOR operations for coding design.
  • Network scheduling with propagation delay
  • Network computation
  • Finite-length coding
